# AI Lifestyle Frontend

React + Material UI frontend for the AI Lifestyle application.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Environment Variables

Create a `.env.local` file:

```
VITE_API_URL=http://localhost:9000
VITE_AUTH_DOMAIN=your-auth-domain
```
